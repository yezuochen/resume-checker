# Resume Checker Skill

**Analyze your resume against any job description**

A comprehensive resume analysis tool that evaluates how well your resume matches a job posting.

## What It Does

5-dimensional analysis: Content, Skills, Format, Sections, Style

## Match Score Weights

| Dimension | Weight |
|-----------|--------|
| Skills    | 40%    |
| Content   | 25%    |
| Style     | 15%    |
| Sections  | 10%    |
| Format    | 10%    |

## Requirements

- Python 3.8+
- reportlab (for PDF generation)
